# ProxManagerGoReact
A remake of ProxMox Manager in Go for the Backend and React for Frontend with the deployment planded to be in Docker or a LXC Conteiner.
<br>
This project is intented to be a Learning expirience, as my first project in Go.

<br><hr><hr><br>

# Go Install
- install go (thats it...)

### Go Libraries
```bash
# Gin
go get github.com/gin-gonic/gin
# Swagger
go get github.com/swaggo/swag@latest
go get github.com/swaggo/gin-swagger
go get github.com/swaggo/files
```

# Run as Dev
Open CMD in Backend folder and run the commands to rebuild Swagger and run the Go API Server.

### Re Create Swagger
```bash
swag init
```

### Run Go Server
```bash
go run . 
```

### Swagger
```bash
http://localhost:8080/swagger/index.html
```
